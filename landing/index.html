Install

Choose your TV platform

Both versions use the same phone-based playlist setup.

Android

Android TV

  1. View releases. The APK appears after a release is published. Check Build status.
  2. Transfer the APK to your Android TV.
  3. Allow installation from your chosen file source if prompted.
  4. Install and open IPTV QR.
  5. Complete QR setup with your phone.
Samsung

Samsung Tizen

  1. Install TizenBrew.
  2. Open TizenBrew on your TV.
  3. Open the Module Manager tab at the top.
  4. Add tizenbrew-iptv.
  5. Complete QR setup.

Before you install

Does IPTV QR include a playlist?

No. You need your own compatible M3U playlist and permission to view its streams.

How does QR setup work?

The TV shows a setup code. Scan it with your phone, then submit your playlist URL from the phone.

Which platforms are supported?

The project provides an Android TV APK and a TizenBrew module for compatible Samsung Tizen TVs.

What should I know about privacy?

The project does not provide or host channel lists. Review the open-source code and your playlist provider's policy before use.
